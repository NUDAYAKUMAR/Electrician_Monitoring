# ⚡ Electrician Contractor Management System

## 📌 Description

This project is a web-based application designed to manage electricians, assign jobs, and track work progress efficiently. It helps admins monitor tasks, electricians complete assigned work, and users request services.

---

## 🚀 Features

* 🔐 Role-based login (Admin, Electrician, User)
* 📊 Admin dashboard with full access
* 👷 Electrician task management
* 📝 Job assignment and tracking
* 📦 Material and task monitoring

---

## 🛠 Tech Stack

* **Frontend:** HTML, CSS, JavaScript
* **Backend:** Flask (Python)
* **Database:** MongoDB

---

## ▶️ How to Run the Project

1. Clone the repository:

   ```bash
   git clone https://github.com/divyasumpagi-sketch/electrician-management-system.git
   ```

2. Navigate to the project folder:

   ```bash
   cd electrician-management-system
   ```

3. Install dependencies:

   ```bash
   pip install -r requirements.txt
   ```

4. Run the application:

   ```bash
   python app.py
   ```

5. Open in browser:

   ```
   http://127.0.0.1:5000
   ```
